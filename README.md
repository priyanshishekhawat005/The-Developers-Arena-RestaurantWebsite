## Project Overview
This is a complete responsive restaurant website for **Saffron Table**.

Implemented features:
- 4 HTML pages (`index.html`, `menu.html`, `booking.html`, `contact.html`)
- Responsive mobile-first layout
- Contact and booking forms with JavaScript validation
- Menu filtering by category
- Accessibility basics (semantic HTML, skip link, focus states, aria-live messages)
